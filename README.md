# Student Performance Analysis Using EDA

## Project Overview

This project explores the factors influencing student academic performance through Exploratory Data Analysis (EDA) and statistical testing. The analysis focuses on identifying key academic, behavioral, and socioeconomic variables that impact exam scores and provides data-driven recommendations for improving educational outcomes.



## Objectives

* Analyze student performance data and understand feature relationships.
* Identify the major factors affecting exam scores.
* Perform statistical testing to validate findings.
* Generate actionable insights and recommendations.
* Demonstrate end-to-end data analysis skills using Python.



## Dataset Information

* **Dataset:** Student Performance Factors
* **Records:** 6,600+ students
* **Target Variable:** Exam Score
* **Data Type:** Structured tabular dataset containing academic, behavioral, and demographic features.

## Key Findings

* Attendance showed one of the strongest positive relationships with exam scores.
* Students with higher study hours generally achieved better academic results.
* Motivation level significantly influenced performance.
* Educational support such as tutoring sessions contributed positively to student outcomes.
* Previous academic performance was a strong indicator of future success.



## Statistical Analysis

The following statistical techniques were used:

* Pearson Correlation Analysis
* Independent T-Test
* One-Way ANOVA

These tests helped validate the significance of relationships observed during exploratory analysis.



## Recommendations

### For Schools

* Improve attendance monitoring systems.
* Expand tutoring and academic support programs.
* Increase access to educational resources.

### For Students

* Maintain consistent attendance.
* Follow structured study schedules.
* Utilize available academic support services.



## Limitations

* Results are based on the available dataset and may not represent all educational environments.
* Some variables may not capture all external factors affecting student performance.
* Correlation does not imply causation.
